//============================================================================== // CSS针对ie10、ie11专有属性 //============================================================================== CSS的媒体查询语句(media query)是一种高级的CSS条件语句，它能根据一些属性和属性值计算判断CSS是否可以生效。在这里，我们要使用一个IE10/11独有的属性，它就是-ms-high-contrast，只有IE10/11实现了这个属性，它可以有两个值active或none，使用下面的媒体查询语句： /* 以下代码仅对ie10、ie11浏览器起作用 */ /* IE10+ CSS styles go here*/ @media all and (-ms-high-contrast: none), (-ms-high-contrast: active) { } //============================================================================== // CSS针对ie8、ie9 //============================================================================== 【关于\9和\0】 IE8、IE9 都能读懂\9和\0 IE11 仅能读懂\9标识 eg1. .content{ background-color: #FF0000; /* 红色 */ background-color: #0000FF\0; /* 蓝色 */ background-color: #FFFF00\9; /* 黄色 */ } 结果：IE8 IE9 IE11均显示黄色 eg2. .content{ background-color: #FF0000; /* 红色 */ background-color: #FFFF00\9; /* 黄色 */ background-color: #0000FF\0; /* 蓝色 */ } 结果：IE11显示黄色，IE8 IE9 均显示蓝色 【ie兼容rotate旋转】 img{ transform: rotate(90deg); filter: progid:DXImageTransform.Microsoft.BasicImage(rotation=90); /*ie8/ } //============================================================================== // CSS媒体查询断点 //==============================================================================
