IMG 标签新属性

IMG N倍（1倍、2倍、3倍）图的使用
方法1：使用CSS媒体查询
/*默认1倍图*/
.photo {
    width: 100px; 
    height: 100px;
    background-image: url("../img/cat.png");
    background-size: 30px auto;
}
/*2倍图*/
@media only screen and (-webkit-min-device-pixel-ratio: 2),
       only screen and (min--moz-device-pixel-ratio: 2),
       only screen and (min-resolution: 240dpi) {
    .photo{
        background-image: url("../img/cat@2x.png");
        background-size: 60px auto;
    }
}
/*3倍图*/
@media only screen and (-webkit-min-device-pixel-ratio: 3),
       only screen and (min--moz-device-pixel-ratio: 3),
       only screen and (min-resolution: 360dpi) {
    .photo{
        background-image: url("../img/cat@3x.png");
        background-size: 90px auto;
    }
}
方法2：使用 IMG 的 SRCSET 属性
    <img width="100" height="100" src="cat.png" srcset="cat@2x.png 2x,cat@3x.png 3x">
