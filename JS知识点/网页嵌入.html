网页嵌入的几个方法

一、任何人都无法嵌入你的网页
            // 写法一：
            if (top.location != self.location) { 
                top.location = self.location; 
            }
            // 写法二：
            if(top != window){
                top.location.href = window.location.href;
            }
二、你的网页可被当前域名的其它网页嵌入，但别人无法嵌入你的网页
            try{
                top.location.hostname;
                if(top.location.hostname != window.location.hostname){ // 兼容谷歌浏览器
                    top.location.href = window.location.href;
                }
            }
            catch{
                top.location.href = window.location.href;
            }
