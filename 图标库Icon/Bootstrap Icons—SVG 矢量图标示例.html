使用示例
方法一、使用CSS ClassName属性，并引入图标的样式文件
说明：可使用CSS的fontSize或Color改变图标外观
<link type="text/css" rel="stylesheet" href="https://icons.getbootstrap.com/font/bootstrap-icons.css"><!--Bootstrap图标-->

.fs-doc{ 
    font-size: 2rem; 
    color: cornflowerblue; 
}
